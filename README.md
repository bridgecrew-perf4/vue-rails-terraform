# 運動ログアプリ

ダイエット運動や食事のログアプリを作成しました。

Docker-compose のビルド
`make build`

Docker イメージの起動
`make up`

マイグレーション
`make migrate`

マスターデータ投入
`make seed`

フロントエンド
localhost:8000

バックエンド
localhost:3000
